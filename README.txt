
MicroBase

Simple PHP and MySQL develop environment


Characteristics:
	version 1.3.2
	last mod: May 1, 2018
	max num tables: 3
	max num items in table: 16



Requeriments (Debian):
    Necessary:
        php
        php-mysql
        mysql-server
        php-xml
        libapache2-mod-php
        apache2

 

Requeriments (FreeBSD):
    Necessary:
        php
        php56-mysqli
        mysql56-server
        php56-simplexml
        mod_php56
        apache24

 

Requeriments (Windows):
    Necessary:
        xampp
    Recommendable:
        binary php in the PATH



Use:
	1. from terminal, run:  php GO.php "My project"   to create 'My project/SCHEME.xml'
	2. edit  'My project/SCHEME.xml'  and configure your database and tables
	3. from terminal, run:  php FILES.php "My project"   to create php files
	4. go to  'My project/install.php'  with a browser to install project
